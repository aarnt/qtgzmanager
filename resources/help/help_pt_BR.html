QTGZManager

Um aplicativo Qt4 para a gerência de pacotes do sistema operacional Slackware, licenciado sob os termos da GPL v2. Para maiores informações, visite o sítio do QTGZManager.


Classificação dos pacotes:
  • Versão inferior do pacote instalado
  • Versão superior do pacote instalado
  • Arquitetura diversa ao pacote instalado
  • Pacote instalado
  • Pacote não instalado
  • Pacote congelado
  • Pacote RPM
Principais comandos:
  • Use os menus de contexto e "arrastar e soltar" para gerar as ações sobre pacotes
  • Selecione "Exibir informações" ou F8 sobre um pacote para ver suas informações
  • Selecione "Exibir conteúdo", pressione Enter ou duplo clique em um pacote para ver seu conteúdo
  • Selecione "Congelar pacote" sobre um pacote instalado para congelar sua versão
  • Selecione "Diff" sobre uma versão antiga/nova de um pacote para ver as mudanças na lista de arquivos
Sequências Control+tecla
  • Ctrl+C / Ctrl+X para copiar / colar pacotes selecionados
  • Ctrl+E ou selecione "Executar ações" para iniciar a execução das ações sobre pacotes
  • Ctrl+F para encontrar um arquivo na lista de pacotes instalados ou no pacote que estiver aberto na visão inferior
  • Ctrl+K para encontrar um pacote no diretório selecionado
  • Ctrl+P para buscar as últimas patches oficiais do Slackware
  • Ctrl+Q para sair da aplicação
  • Ctrl+T para entrar em Configurações
  • Ctrl+S para salvar um instantâneo da lista de pacotes instalados
  • Ctrl+V para colar pacotes dentro de um diretório
Sequências Control+shift+tecla
  • Ctrl+Shift+D para ir para o diretório padrão
  • Ctrl+Shift+F para encontrar um arquivo dentro do pacote selecionado
  • Ctrl+Shift+K para ir para o diretório cache do Slackpkg
  • Ctrl+Shift+P para ir para o diretório de patches do Slackware
  • Ctrl+Shift+S para ir para o diretório /tmp/SBo
  • Ctrl+Shift+T para ir para o diretório /tmp
  • Ctrl+Shift+V para ir para o diretório /var
Sequências F+tecla:
  • F1 para visualizar esta ajuda
  • F2 para renomear o diretório selecionado
  • F4 para abrir um Terminal no diretório selecionado
  • F6 para abrir um Gerenciador de Arquivos no diretório selecionado
  • F8 para ver as informações do pacote
  • F9 para exibir/esconder os pacotes instalados (visão da direita)
  • F10 para exibir as 4 visões (visão normal)
  • F11 para minimizar a visão inferior
  • F12 para maximizar a visão inferior
