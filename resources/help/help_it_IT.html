QTGZManager

Un'applicazione scritta in Qt4 per la gestione dei pacchetti Slackware, applicazione rilasciata sotto i termini della GPL v2 Per maggiori informazioni visita il sito web di QTGZManage


Classificazione Pacchetti:
  • Versione inferiore di un pacchetto installato
  • Versione superiore di un pacchetto installato
  • Differente architettura di un pacchetto installato
  • Pacchetto installato
  • Pacchetto non installato
  • Pacchetto bloccato
  • Pacchetto RPM
Aiuto di base:
  • Usa il drag and drop oppure i menu contestuali per generare azioni
  • Seleziona "Visualizza informazioni" su di un pacchetto per averne informazioni (Oppure premi F8)
  • Seleziona "Visualizza contenuto" su di un pacchetto per vederne il contenuto (Oppure premi Enter/Doppio click)
  • Seleziona "Blocca pacchetto" su di un pacchetto installato per bloccarne la versione
  • Seleziona "Differenza" su di una vecchia/nuova versione di un pacchetto per vederne la lista dei file cambiati
Control+key sequenze:
  • Ctrl+C / Ctrl+X Per copiare o tagliare i pacchetti selezionati
  • Ctrl+E Seleziona "Esegui" per avviare l'esecuzione delle azioni programmate
  • Ctrl+F Per cercare un file nei pacchetti installati oppure nel pacchetto mostrato nella finestra in basso
  • Ctrl+K Per cercare un pacchetto nella directory selezionata
  • Ctrl+P Per controllare le patches ufficiali Slackware
  • Ctrl+Q Per uscire dall'applicazione
  • Ctrl+S Per creare un'istantanea dei pacchetti installati
  • Ctrl+T Per entrare nelle impostazioni
  • Ctrl+V Per incollare i pacchetti nella directory
Control+shift+key sequenze:
  • Ctrl+Shift+D Per andare nella directory di default
  • Ctrl+Shift+F Per cercare un file nel pacchetto selezionato
  • Ctrl+Shift+K Per andare nella cache della directory di Slackpkg
  • Ctrl+Shift+P Per andare nella cache della directory delle patch Slackware
  • Ctrl+Shift+S Per andare nella directory /tmp/SBo
  • Ctrl+Shift+T Per andare nella directory /tmp
  • Ctrl+Shift+V Per andare nella directory /var
F+key sequenze:
  • F1 Per mostrare quest'aiuto
  • F2 Per rinominare le directory selezionate
  • F4 Per aprire un Terminale nella directory selezionata
  • F6 Per aprire un File Manager nella directory selezionata
  • F8 Per avere informazioni sul pacchetto
  • "F9" Per mostrare/nascondere la finestra dei Pacchetti Installati (Visuale destra)
  • "F10" Per mostrare le 4 finestre di visualizzazione (Visualizzazione normale)
  • "F11" Per nascondere la finestra di visualizzazione bassa
  • "F12" Per mostrare la finestra di visualizzazione bassa
