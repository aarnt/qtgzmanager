QTGZManager

Ένα διαχείριστής πακέτων για το Slackware βασισμένος στο Qt4 Άδεια σύμφωνα με τους όρους της GPL v2. Για περισσότερες πληροφορίες επισκεφτείτε το QTGZManager's κεντρική ιστοσελίδα.


Ταξινόμηση πακέτων:
  • Inferior έκδοση ενός εγκατεστημένου πακέτου
  • Superior έκδοση ενός εγκατεστημένου πακέτου
  • Διαφορετική arch ενός εγκατεστημένου πακέτου
  • Eγκατεστημένο πακέτο
  • Μη εγκατεστημένο πακέτο
  • Παγωμένο πακέτο
  • RPM πακέτο
Βασική βοήθεια χρήσης:
  • Χρησιμοποιήστε την μεταφορά και απόθεση ή το μενού περιβάλλοντος για τη δημιουργία ενεργειών του πακέτου
  • Επιλέξτε «Προβολή πληροφοριών» ή F8 πάνω από ένα πακέτο για να δείτε τις πληροφορίες του
  • Επιλέξτε «Προβολή περιεχομένου» ή πατήστε το πλήκτρο Enter ή κάντε διπλό κλικ σε ένα πακέτο για να δείτε τις πληροφορίες του
  • Επιλέξτε «Πάγωμα πακέτου» πάνω από ένα εγκατεστημένο πακέτο για να παγώσετε την έκδοση του
  • Επιλέξτε «diff» πάνω από μια παλιά/νέα έκδοση ενός πακέτου για να δείτε τι έχει αλλάξει
Control+key συνδυασμοί:
  • Ctrl+C / Ctrl+X για να αντιγράψετε / για να αποκόψετε το επιλεγμένο πακέτο(α)
  • Ctrl+E ή επιλέξτε «Εκτέλεση» για να ξεκινήσει η εκτέλεση των ενεργειών
  • Ctrl+F για να βρείτε ένα αρχείο σε εγκατεστημένα πακέτα ή σε πακέτα που φαίνονται απο κάτω
  • Ctrl+K για να βρείτε ένα πακέτο στον επιλεγμένο φάκελο
  • Ctrl+P για να ελέγξετε για επίσημα patche του Slackware
  • Ctrl+Q για να βγείτε από την εφαρμογή
  • Ctrl+S για να πάρετε ένα στιγμιότυπο από τα εγκατεστημένα πακέτα
  • Ctrl+T για είσοδο στις ρυθμίσεις
  • Ctrl+V για να επικολλήσετε τα πακέτα σε έναν φάκελο
Control+shift+key συνδυασμοί:
  • Ctrl+Shift+D για να πάτε στον προεπιλεγμένο φάκελο
  • Ctrl+Shift+F για να βρείτε ένα αρχείο μέσα στο επιλεγμένο πακέτο
  • Ctrl+Shift+K για να πάτε στον φάκελο της cache του Slackpkg
  • Ctrl+Shift+P για να πάτε στον φάκελο των patch του Slackware
  • Ctrl+Shift+S για να πάτε στον φάκελο /tmp/SBo
  • Ctrl+Shift+T για να πάτε στον φάκελο /tmp
  • Ctrl+Shift+V για να πάτε στον φάκελο /var
F+key συνδυασμοί:
  • F1 για να δείτε αυτή τη βοήθεια
  • F2 για να μετονομάσετε τον επιλεγμένο φάκελο
  • F4 για να ανοίξετε ένα τερματικό στον επιλεγμένο φάκελο
  • F6 για να ανοίξετε ένα διαχειριστή αρχείων στον επιλεγμένο φάκελο
  • F8 για να δείτε πληροφορίες για το πακέτο
  • F9 για να εμφανίσετε/αποκρύψετε την προβολή των εγκατεστημένων πακέτων (δεξιά προβολή)
  • F10 για να δείτε και τις 4 εμφανίσεις (κανονική προβολή)
  • F11 για να ελαχιστοποιήσετε την κάτω προβολή
  • F12 για να μεγιστοποιήσετε την κάτω προβολή
