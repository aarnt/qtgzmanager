QTGZManager

A Qt4-based Slackware's package management application, licensed under the terms of GPL v2. For more information visit QTGZManager's main site.


Package classification:
  • Inferior version of an installed package
  • Superior version of an installed package
  • Different arch of an installed package
  • Installed package
  • Non installed package
  • Frozen package
  • RPM package
Basic usage help:
  • Use drag and drop or context menus to generate package actions
  • Select "View information" or F8 over a package to see it's info
  • Select "View content" or press Enter or double click a package to see it's content
  • Select "Freeze package" over an installed package to freeze it's version
  • Select "Diff" over an old/new version of a package to see what have changed
Control+key sequences:
  • Ctrl+C / Ctrl+X to copy / cut selected package(s)
  • Ctrl+E or select "Execute" to start the execution of actions
  • Ctrl+F to find a file in installed packages or in package shown at lower view
  • Ctrl+K to find a package in selected directory
  • Ctrl+P to check for official Slackware patches
  • Ctrl+Q to exit the application
  • Ctrl+S to take a snapshot of installed packages
  • Ctrl+T to enter Setup
  • Ctrl+V to paste packages into a directory
Control+shift+key sequences:
  • Ctrl+Shift+D to go to default directory
  • Ctrl+Shift+F to find a file inside selected package
  • Ctrl+Shift+K to go to Slackpkg cache directory
  • Ctrl+Shift+P to go to Slackware patches directory
  • Ctrl+Shift+S to go to /tmp/SBo directory
  • Ctrl+Shift+T to go to /tmp directory
  • Ctrl+Shift+V to go to /var directory
F+key sequences:
  • F1 to show this help
  • F2 to rename selected directory
  • F4 to open a Terminal in selected directory
  • F6 to open a File Manager in selected directory
  • F8 to see package info
  • F9 to show/hide Installed Packages view (right view)
  • F10 to show all 4 views (normal view)
  • F11 to minimize Lower view
  • F12 to maximize Lower view
